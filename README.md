# webtoolkit
Gulp toolkit to quickly and easily set up a Single Page Application

## Initialization

```bash
yarn add --dev @yodas/webtoolkit
```

Add to your `gulpfile.js`:
```javascript
require('node_modules/@yodasws/webtoolkit/gulpfile.js');
```

```bash
gulp init
```

## Tasks

```bash
gulp -?
```
